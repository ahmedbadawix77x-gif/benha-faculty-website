# 🎓 Faculty of Specific Education – Benha University

### Interactive Promotional Website

A modern, high-end, interactive university website designed to showcase the Faculty of Specific Education at Benha University.

The project focuses on delivering a **cinematic user experience**, combining clean academic design with smooth animations to create a visually engaging and professional platform.

---

## 🎯 Project Goals

* Present all faculty departments in a modern way
* Attract new students through visual storytelling
* Highlight academic programs and career opportunities
* Deliver a premium UI/UX experience

---

## 🧩 Features

* ✨ Smooth animations and transitions
* 🎬 Interactive sections with modern UI
* 🏛️ Dedicated pages for each department
* 🎥 Video showcase (Dean & Department Heads)
* 📱 Fully responsive design
* ⚡ Fast and optimized performance

---

## 🛠️ Run Locally

### Prerequisites:

* Node.js installed

### Steps:

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local` file and add your API key:

```env
GEMINI_API_KEY=your_api_key_here
```

3. Run the project:

```bash
npm run dev
```
